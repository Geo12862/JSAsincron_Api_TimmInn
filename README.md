## Formularios
### Formato Contraseña:
- Primera letra Mayuscula, segunda letra minuscula, de 4 a 8 letras minusculas o mayusculas y 2 números
- Ejemplos: Aabbbb11, AaBBBB99, Aabbbbbbbb11, AaBBBBBBBB99...
